# mit-load repository
## I denne mappen ligger læringsressurser for hvordan marsj med oppakning påvirker soldatens energiforbruk og fysiskeprestasjonsevne / strdsevne.
## Læringsmodulen er ment for å gi kadetter en innsikt i og hvordan soldaten kan imøtekomme arbeidskravet marsj med oppakning på en hensiktsmessig måte.
